CELLAR media types

The Publications Office of the EU uses media types as defined in the IANA MIME Media Type registry: http://www.iana.org/assignments/media-types . In case a media type is not registered (yet) in the IANA registry, the media type defined by the vendor or developer of the media type is used instead. In case of multiple media types with the same file extension, the media type is completed with a sub-type specifying the exact media type. For example: application/pdf; sub-type="pdfa-1a".
The following media types (IANA registered and unregistered) are permitted in the CELLAR:
