REF.APPEAL.TABLE

[element]

Link to the table of appeals in an ECR publication

The REF.APPEAL.TABLE element is used to mark up a link to the list of appeals in a case.

Model

<xd:element name="REF.APPEAL.TABLE" >
  <xd:complexType >
    <xd:sequence >
      <xd:element ref="TITLE" />
      <xd:element ref="REF.ECR.DOC" />
    </xd:sequence>
  </xd:complexType>
</xd:element>

Used by

ECR

General rules

Element

It contains the title of the list of appeals (TITLE) and the reference to the case (REF.ECR.DOC).
