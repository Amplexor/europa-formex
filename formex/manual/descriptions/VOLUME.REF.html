VOLUME.REF

[element]

Reference to a volume

The VOLUME.REF element is used within the bibliographical description of documents in order to mark up the identifier of the volume to which the document belongs.

Model

<xd:element name="VOLUME.REF" >
  <xd:simpleType >
    <xd:restriction base="xd:string" >
      <xd:pattern value="\d{2}" />
    </xd:restriction>
  </xd:simpleType>
</xd:element>

Used by

DOCUMENT.REF DOCUMENT.REF.CONS PUBLICATION.REF

General rules

Element

The volume number always consists of two digits. If necessary, the initial position must be filled with zero.
