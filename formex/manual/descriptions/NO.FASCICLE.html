NO.FASCICLE

[element]

Fascicle number

The NO.FASCICLE element is used to mark up the fascicle number within the volume of an ECR publication.

Model

<xd:element name="NO.FASCICLE" type="xd:string" />

Used by

BIB.ECR NO.FASCICLE

General rules

Element

The element must only contain character data. No sub-elements are allowed. Information on double fascicles and on parts such as A or B has to be included.
