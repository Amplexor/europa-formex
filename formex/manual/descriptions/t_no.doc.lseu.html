t_no.doc.lseu

[simple type]

Identifier of a legislation summary

This simple type constructs the identifier of a legislation summar.

Model

<xd:simpleType name="t_no.doc.lseu" >
  <xd:restriction base="xd:string" >
    <xd:pattern value="LSEU\-((GLOSSARY\-[A-Z0-9_]+)|(INTRO\-(\d{2})+)|([A-Z]([A-Z]|\d)\d{4}[A-Z]?)|(\d{2}(\d{2}){0,5}_[1-9]\d{0,3})|([4-9](\d{4}|\d{6})))\-\d{4}\-\d{2}\-\d{2}\-\d{4}" />
  </xd:restriction>
</xd:simpleType>

General rules

Type

The simple type is based on xs:string and composed by

  • a prefix with the fixed value 'LSEU',
  • the resource type (INTRO, GLOSSARY) if different from SUMMARY,
  • the identifier of the domain, either alfanumeric or numeric,
  • the work id,
  • four digits representing the year of the production,
  • two digits representing the month of the production,
  • two digits for the day of the production,
  • four digits as a sequential number of production at a given date.

The value is identical for all language versions of a summary.
