NO.OJ

[element]

OJ number

The NO.OJ element is used to mark up the OJ number.

Throughout a publication year, each copy of the OJ is identified by a unique number, which is specific to the collection. The first copy of the year is given the number 1, the second the number 2, etc.

Model

<xd:element name="NO.OJ" type="t_no.oj" />

Used by

BIB.OJ DOC.CORR DOCUMENT.REF DOCUMENT.REF.CONS NO.OJ OJ.CL PUBLICATION.REF REF.OJ SPEC.ED

General rules

Element

The OJ number is composed according to the rules defined in the t_no.oj simple type. It always consists of a numerical value, optionally followed by a letter for specific collections.

Depending on the OJ collection, the letter 'A' (Annex), the letter 'E' (Electronic), or 'M' (Maltese) must be added to the OJ number.

Example

<COLL>L</COLL>
 <NO.OJ>010</NO.OJ>
<COLL>C</COLL>
 <NO.OJ>145E</NO.OJ>
