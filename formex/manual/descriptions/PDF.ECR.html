PDF.ECR

[element]

References to the PDF instances created for a case

The PDF.ECR element is used to mark up the references to the PDF instances which have been created for a case.

Model

<xd:element name="PDF.ECR" >
  <xd:complexType >
    <xd:sequence >
      <xd:element ref="REF.PDF.ECR" maxOccurs="unbounded" />
    </xd:sequence>
  </xd:complexType>
</xd:element>

Used by

CASE

General rules

Element

It contains one or more REF.PDF.ECR elements.

Example

<PDF.ECR>
  <REF.PDF.ECR TYPE="SUMMARY" PAGE.FIRST="00439" PAGE.LAST="00440">//filename//</REF.PDF.ECR>
  <REF.PDF.ECR TYPE="CONCLUSION" PAGE.FIRST="00441" PAGE.LAST="00457">//filename//</REF.PDF.ECR>
  <REF.PDF.ECR TYPE="CASE" PAGE.FIRST="00458" PAGE.LAST="00480">//filename//</REF.PDF.ECR>
 </PDF.ECR>
