PAPER.GEN

[element]

Description of a paper issue (general publication)

The PAPER.GEN element is used to mark up the paper issue of a general publication.

Model

<xd:element name="PAPER.GEN" >
  <xd:complexType >
    <xd:sequence >
      <xd:element ref="TITLE" minOccurs="0" />
      <xd:element ref="VOLUME.PAPER" maxOccurs="unbounded" />
    </xd:sequence>
  </xd:complexType>
</xd:element>

Used by

BIB.GEN.PUB

General rules

Element

The PAPER.GEN element contains information on the presentation of a given document in the summary of the paper publication. It contains an optional TITLE and one or more VOLUME.PAPER elements.
