BIB.OJ

[element]

Bibliographical description

The BIB.OJ element is used to mark up the bibliographical and physical description of an OJ.

Model

<xd:element name="BIB.OJ" >
  <xd:complexType >
    <xd:sequence >
      <xd:element ref="COLL" minOccurs="0" />
      <xd:element ref="TITLE" minOccurs="0" />
      <xd:choice minOccurs="0" >
        <xd:element ref="NO.OJ" />
        <xd:element ref="SPECIAL" />
      </xd:choice>
      <xd:element ref="AGE.OJ" minOccurs="0" />
      <xd:element ref="DATE" minOccurs="0" />
      <xd:element ref="LAST.OJ" minOccurs="0" />
      <xd:element ref="LG.OJ" />
      <xd:element ref="PRINTER" />
      <xd:element ref="REF.CORE.METADATA" minOccurs="0" />
      <xd:element ref="REF.BIB.RECORD" minOccurs="0" />
    </xd:sequence>
    <xd:attribute name="REPUBLICATION" type="t_boolean" default="NO" />
  </xd:complexType>
</xd:element>

Used by

OJ

General rules

Element

The BIB.OJ provides the following bibliographical informations:

  • the collection to which the publication belongs (COLL),
  • the title of the publication (TITLE),
  • the OJ number (NO.OJ),
  • in the context of special editions, for the time being in Irish and Maltese, the NO.OJ element is replaced by SPECIAL,
  • the number of years of publication of the OJ (AGE.OJ),
  • the date of publication (DATE),
  • the indication of the last publication of the year (LAST.OJ),
  • the language in which the publication appears (LG.OJ),
  • the code of the printer responsible for publishing and printing the publication (PRINTER),
  • optionally a reference to the core metadata (REF.CORE.METADATA),
  • optionally a reference to a bibliographic record (REF.BIB.RECORD).

These sub-elements may be empty, if their value is unknown or not yet fixed during the pre-press phase.

All elements are optional, apart from the LG.OJ and the PRINTER elements, which are mandatory.

When the publication is a secondary legislation publication, the bibliographical information is described using the BIB.SE element.

Example

<BIB.OJ>
  <COLL>L</COLL>
  <NO.OJ CLASS="NORMAL">73</NO.OJ>
  <DATE ISO="20020315">20020315</DATE>
  <AGE.OJ>45</AGE.OJ>
  <LG>EN</LG>
  <PRINTER>OP</PRINTER>
 </BIB.OJ>
