OBJECT

[element]

Object of a case

The OBJECT element is used to mark up the object of a case.

Model

<xd:element name="OBJECT" >
  <xd:complexType >
    <xd:sequence >
      <xd:element ref="TITLE" />
      <xd:element ref="P" />
    </xd:sequence>
  </xd:complexType>
</xd:element>

Used by

CASE

General rules

Element

The element contains a title (TITLE) followed by a P element.
