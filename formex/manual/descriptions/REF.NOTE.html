REF.NOTE

[element]

Reference to a note

The REF.NOTE element is used to mark up the reference to a note, which may appear at the end of a structured text block.

Model

<xd:element name="REF.NOTE" >
  <xd:complexType >
    <xd:attribute name="REF" type="xd:IDREF" use="required" />
  </xd:complexType>
</xd:element>

Used by

LOC.NOTES

General rules

Element

The REF.NOTE element is used within LOC.NOTES in order to mark up each reference to a note. It does not contain any sub-elements. The information is encoded within the mandatory attribute (REF).

Attributes

The REF attribute

NOTE

Example

In the following example, two notes are called in a text block structure. Their contents have to be located at the end of the highest-level GR.SEQ :

<GR.SEQ LEVEL="1">
  <NOTE ID="E0001" TYPE="MANUAL">...</NOTE>
  <GR.SEQ LEVEL="2">
    <NOTE ID="E0002" TYPE="MANUAL">...</NOTE>
  </GR.SEQ>
  <GR.SEQ LEVEL="2">  </GR.SEQ>
  <LOC.NOTES>
    <REF.NOTE REF="E0001">    </REF.NOTE>
    <REF.NOTE REF="E0002">    </REF.NOTE>
  </LOC.NOTES>
 </GR.SEQ>
