VOLUME.ECR

[element]

Volume number of an ECR publication

The VOLUME.ECR element is used to mark up the volume number of an ECR publication.

Model

<xd:element name="VOLUME.ECR" type="xd:string" />

Used by

REF.ECR VOLUME.ECR

General rules

Element

The element must only contain character data. No sub-elements are allowed.

Example

<VOLUME.ECR>10</VOLUME.ECR>
