NO.CASE

[element]

Number of a case

The NO.CASE element is used to mark up the number of a case.

Model

<xd:element name="NO.CASE" type="xd:string" />

Used by

t_btx.seq BIB.CASE NO.CASE REF.CASE REF.CASE.F1 REF.DOC.ECR

General rules

Element

The element must only contain character data. No sub-elements are allowed.

In addition to that restriction no typographical characters such as no-break-spaces, tabulation characters etc. may be used.
