DOCUMENT.REF.CONS

[element]

Reference to the document instance in the context of consolidation

The DOCUMENT.REF.CONS element contains the reference to the document instance in the context of consolidation.

Model

<xd:element name="DOCUMENT.REF.CONS" >
  <xd:complexType >
    <xd:sequence >
      <xd:element ref="COLL" minOccurs="0" />
      <xd:element ref="NO.OJ" minOccurs="0" />
      <xd:element ref="YEAR" minOccurs="0" />
      <xd:element ref="LG.OJ" minOccurs="0" />
      <xd:element ref="PAGE.FIRST" minOccurs="0" />
      <xd:element ref="PAGE.SEQ" minOccurs="0" />
      <xd:element ref="VOLUME.REF" minOccurs="0" />
    </xd:sequence>
  </xd:complexType>
</xd:element>

Used by

BIB.INSTANCE.CONS

General rules

Element

The DOCUMENT.REF.CONS element contains the following optional sub-elements:

  • the collection to which the document belongs (COLL),
  • the OJ number to which the document belongs (NO.OJ),
  • the year when the document was published (YEAR),
  • the language in which the publication appears (LG.OJ),
  • the page number on which the document begins (PAGE.FIRST),
  • the sequence number of the document on the first page (PAGE.SEQ),
  • the reference to the volume to which the document belongs (VOLUME.REF),

Example

<BIB.INSTANCE.CONS>
  <DOCUMENT.REF.CONS>
    <COLL>L</COLL>
    <NO.OJ>113</NO.OJ>
    <YEAR>2001</YEAR>
    <LG.OJ>EN</LG.OJ>
    <PAGE.FIRST>14</PAGE.FIRST>
    <PAGE.SEQ>1</PAGE.SEQ>
    <VOLUME.REF>01</VOLUME.REF>
  </DOCUMENT.REF.CONS>
  <DATE ISO="20010425">20010425</DATE>
  <LG.DOC>EN</LG.DOC>
  <PAGE.FIRST>14</PAGE.FIRST>
  <PAGE.SEQ>1</PAGE.SEQ>
  <PAGE.LAST>14</PAGE.LAST>
  <PAGE.TOTAL>1</PAGE.TOTAL>
 </BIB.INSTANCE.CONS>
