REF.RULING

[element]

Reference to a ruling

The REF.RULING element is used to mark up the reference to an instance which contains a ruling.

Model

<xd:element name="REF.RULING" >
  <xd:complexType >
    <xd:sequence >
      <xd:element ref="TITLE" />
      <xd:element ref="REF.ECR.DOC" />
    </xd:sequence>
  </xd:complexType>
</xd:element>

Used by

CASE

General rules

Element

It contains the title of the ruling (TITLE) and the reference to the instance (REF.ECR.DOC).

Both elements are mandatory.

Example

<REF.RULING>
  <TITLE>
    <TI>
      <P>
        <HT TYPE="UC">Délibération de la Cour du
          <DATE ISO="19781114">14 novembre 1978</DATE>
 </HT>
      </P>
    </TI>
  </TITLE>
  <REF.ECR.DOC FILE="ECR1978FR.010215101.xml" NO.SEQ="0001.0001">  </REF.ECR.DOC>
 </REF.RULING>
