DECISION.ECR.INIT

[element]

Introduction of a judgment

The element is used to mark-up the introduction of a judgment, which generally contains the number of a Court decision.

It may only contain the elements defined in the complex type t_btx.

Model

<xd:element name="DECISION.ECR.INIT" type="t_btx" />

Used by

Example
