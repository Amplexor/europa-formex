REF.DOC

[element]

Definition of a link to another document

The REF.DOC element defines a link to another document in the context of legislation summaries.

Model

<xd:element name="REF.DOC" >
  <xd:complexType mixed="true" >
    <xd:complexContent >
      <xd:extension base="t_btx.seq" >
        <xd:attribute name="URI" type="xd:anyURI" use="required" />
        <xd:attribute name="CONTEXT" use="required" >
          <xd:simpleType >
            <xd:restriction base="xd:string" >
              <xd:enumeration value="EUR-LEX" />
              <xd:enumeration value="EXTERNAL" />
              <xd:enumeration value="LSEU" />
            </xd:restriction>
          </xd:simpleType>
        </xd:attribute>
        <xd:attribute name="TITLE" type="xd:string" use="optional" />
      </xd:extension>
    </xd:complexContent>
  </xd:complexType>
</xd:element>

Used by

t_btx t_btx.ecr t_btx.seq

General rules

Element

The content is created on the basis of the the t_btx.seq type..

Attributes

The URI attribute

The URI attribute defines the link in form of a URI.

Attributes

The CONTEXT attribute

  • EUR-LEX: the target of the link is the legal information system EUR-LEX,
  • EXTERNAL: the target is outside a controlled information system,
  • LSEU: the target is another legislation summary.

Attributes

The TITLE attribute
