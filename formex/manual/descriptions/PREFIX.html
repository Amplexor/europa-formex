PREFIX

[element]

Prefix of terms

The PREFIX element is used to mark up the symbol or the number which precedes a term in a list of definitions.

Model

<xd:element name="PREFIX" type="t_btx.seq" />

Used by

DLIST.ITEM PREFIX

General rules

Element

DLIST.ITEM

Example

<DLIST.ITEM>
  <PREFIX>--</PREFIX>
  <TERM>PCE
    <HT TYPE="SUB">red</HT>
 </TERM>
  <DEFINITION>Amount deduced from the risk of future potential credit.</DEFINITION>
 </DLIST.ITEM>
