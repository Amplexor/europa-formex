GR.CORRIG.CL

[element]

Group of corrigenda modifying the basic act

The GR.CORRIG.CL element is used to mark up a group of corrigenda in the context of consolidated acts.

  </xd:complexType>
</xd:element>

General rules

Element

The element has to be used even if it only refers to one object. Therefore it contains an unlimited number of CORRIG.CL elements.
