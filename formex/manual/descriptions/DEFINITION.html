DEFINITION

[element]

Definition of a term

The DEFINITION element is used to mark up the definition associated with a term (TERM) in the context of a list of definitions (DLIST).

Model

<xd:element name="DEFINITION" type="t_btx" />

Used by

DEFINITION DLIST.ITEM

General rules

Element

The DEFINITION element may contain text or any of the elements defined in the complex type t_btx.
