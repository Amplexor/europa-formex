LG.DOC

[element]

Language of a document

The LG.DOC element is used to mark up the language(s) of a document.

Model

<xd:element name="LG.DOC" type="t_language" />

Used by

BIB.CASE BIB.ECR.GENERAL BIB.INSTANCE BIB.INSTANCE.CONS BIB.LSEU DOC.MAIN.PUB DOC.SUB.PUB LG.DOC

General rules

Element

The allowed values, based on ISO 639-1, are listed in the t_language simple type.

This information is provided in the bibliographical description of the document parts (DOC.MAIN.PUB and DOC.SUB.PUB elements), as well as in the instances of the document parts within the bibliographical references (BIB.INSTANCE).

Within the bibliographical description, the language of annexed or related documents (in DOC.SUB.PUB) is given if it is different from the one of the main document (in DOC.MAIN.PUB).
