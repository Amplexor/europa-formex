TOME.SE

[element]

Volume number of a publication in secondary legislation

The TOME.SE element is used to mark up the volume number of a publication relating to the secondary legislation.

General rules

The TOME.SE element is used within the SPEC.ED element used to identify a secondary legislation publication.
