TARIC

[element]

Integrated Tariff of the European Communities

The TARIC element is used to mark up the data relating to the Integrated Tariff of the European Communities. For each TARIC table, the following additional tags may be used:

  • GR.NOTES: optionally used to mark up the text of footnotes.
  • DEF.HEAD.TA: used to mark up the column header.
  • UNIT.TA: used to mark up the description of goods.
