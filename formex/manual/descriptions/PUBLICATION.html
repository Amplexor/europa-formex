PUBLICATION

[element]

Bibliographical and physical description of a publication

The PUBLICATION element is the root element for instances describing a publication.

The L and C collection of the OJ, the special editions (secondary legislation), as well as more general publications are covered by this element.

It is used to identify all the bibliographical elements and the documents contained in the publication.

General rules

Element

The PUBLICATION element consists of two main parts :

  • the bibliographical description: OJ for OJ publications, BIB.GEN.PUB for other publications, SE for special edition publications of the secondary legislation;
  • the content of the publication, which can be structured using SECTION elements. Physical references to the documents belonging to the publication are marked up using a series of ITEM.PUB elements. An illustrative example of this structure is included within the OJ element.
