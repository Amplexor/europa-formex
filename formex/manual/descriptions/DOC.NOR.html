DOC.NOR

[element]

Document without relevance to the consolidation process

The DOC.NOR element is used to mark up a document which has no impact on the consolidation process.

Model

<xd:element name="DOC.NOR" >
  <xd:complexType >
    <xd:attribute name="NO.CELEX" type="xd:string" use="required" />
    <xd:attribute name="COMMENT" type="xd:string" />
  </xd:complexType>
</xd:element>

Used by

GR.DOC.NOR

General rules

Element

The element is empty. The necessary information is specified by attributes.

Attributes

The NO.CELEX attribute

The attribute is mandatory and references the document by its CELEX number.

Attributes

The COMMENT attribute

This attribute allows to add any comment in the document.
