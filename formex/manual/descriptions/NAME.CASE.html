NAME.CASE

[element]

Common name of a case

The NAME.CASE element the commonly used name to refer to a given case.

Model

<xd:element name="NAME.CASE" type="t_btx.seq" />

Used by

NAME.CASE REF.DOC.ECR

General rules

Element

The content of the element is a string which generally is composed of the parties involved in the case.

Example

<NAME.CASE>van Pommeren-Bourgondiën</NAME.CASE>
