REF.REPORT.HEARING

[element]

Reference to a hearing report

The REF.REPORT.HEARING element is used to mark up the reference to an instance which contains a hearing report.

Model

<xd:element name="REF.REPORT.HEARING" >
  <xd:complexType >
    <xd:sequence >
      <xd:element ref="TITLE" />
      <xd:element ref="REF.ECR.DOC" />
    </xd:sequence>
  </xd:complexType>
</xd:element>

Used by

CASE

General rules

Element

It contains the title of the report (TITLE) and the reference to the instance (REF.ECR.DOC).

Both elements are mandatory.

Example

<REF.REPORT.HEARING>
  <TITLE>
    <TI>
      <P>
        <HT TYPE="UC">Rapport d'audience</HT>
      </P>
      <P>présenté dans l'affaire 276/85</P>
    </TI>
  </TITLE>
  <REF.ECR.DOC FILE="ECR1987FR.020049601.xml" NO.SEQ="0001.0001">  </REF.ECR.DOC>
 </REF.REPORT.HEARING>
