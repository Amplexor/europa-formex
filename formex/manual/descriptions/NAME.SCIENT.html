NAME.SCIENT

[element]

Scientific name of the species

The NAME.SCIENT element is used to mark up the scientific name of a vegetable or an agricultural plant species.

Model

<xd:element name="NAME.SCIENT" type="t_btx.seq" />

Used by

NAME.SCIENT TI.SPECIE

General rules

Element

In the publication, the scientific name is presented in upper case characters. In the document with the contents, however, it is presented in lower case characters. To facilitate the generation of this document, the scientific name must be entered in lower case characters.

Since the scientific name is marked up with a specific element which can be used to deduce the way it is presented directly, the HT TYPE='UC' tag is not used in this context.
